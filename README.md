# Binary ancestor

Given a binary tree, find the nearest common ancestor.

```
                  70
         .--------+-------.
         49               84
    .----+---.        .----+---.
   37        54      78        85
 .--+--.    .+    .---+---.
22     40  51    76       80

```

For example.

Ancestor (40, 78) = 70 <br />
Ancestor (51, 37) = 49
